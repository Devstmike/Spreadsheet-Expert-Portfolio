Student Performance Overview

This Excel worksheet contains students scores in three tests. Each test is graded over 100, making the total possible score for all three tests 300. The three test scores is then added together for each student and the student score percentage of the total possible score (300) is calculated. The pass mark is set at 50% which is equivalent to 150 marks. Any student scoring below this mark is considered to have failed. Conditional formatting is used to highlight students who have failed or passed, making it easy to quickly identify their status. Upon summarizing the entire dataset, it shows that 97% of the students have passed while 3% have failed.

Planner

This Excel task planner worksheet accommodates numerous tasks, allowing users to assign both status and priority to each task. Available status options comprise completed, in progress, pending, and not started, while priority choices include high, medium, and low. The sheet features conditional formatting to visually highlight priorities, using red for high, yellow for medium, and a blue-like color for low priority tasks. Additionally, a conditional formatting feature causes the entire row to turn green when the status of a task is marked as completed.
